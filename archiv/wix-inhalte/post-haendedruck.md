# Artikeltext (Dokumentreihenfolge)

Bearbeiten Sie die Gestaltung Ihres Blogs, indem Sie mit der Maus über Ihren Blog-Feed fahren Design auswählen. Hier können Sie unter verschiedenen Layoutvorlagen eine auswählen. Wenn Sie einen weiteren Blog-Feed an einer anderen Stelle auf Ihrer Website hinzufügen, können Sie ein anderes Design-Layout auswählen.

Unter Einstellungen können Sie festlegen, welche Infos und Details angezeigt werden sollen (halten Sie nach den drei Punkten Ausschau).

Im Wix Blog können Sie Autorennamen, Bild, Datum, Lesezeit, Ansichtszähler oder Kommentarzähler beliebig ein- und ausblenden. Wechseln Sie zwischen den Optionen und sehen Sie die Änderungen in Echtzeit.

Sollte Ihr Blog an einen Mitgliederbereich angebunden sein, stellen Sie sicher, dass der Login-Button für all Ihre Nutzer sichtbar ist. Um automatisierte E-Mails zu versenden, klicken Sie in der Bearbeitungsleiste auf E-mail-Benachrichtigungen senden. Sobald Sie zufrieden mit den Einstellungen sind, gehen Sie zu Beiträge verwalten, um Ihre Beiträge weiter zu bearbeiten.
